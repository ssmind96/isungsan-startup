05 · Featured Themes

주요 테마

무료 기반의 Astra와 이미 라이선스를 보유한 Avada를 중심으로 검토합니다.

무료 테마

Astra

가볍고 널리 쓰이는 무료 테마입니다. 기본 설치 후 교회 홈페이지에 맞는 디자인과 기능을 직접 구성해야 합니다.

  • 초기 비용 부담이 낮음
  • 교회 색상·메뉴·메인 화면 커스터마이징 필요
  • 예약·영상·사진 기능은 플러그인 또는 별도 개발로 보완
무료 기반으로 시작하기 좋지만, 완성도 있는 교회 사이트를 위해 세부 화면 구성과 기능 조정 작업이 필요합니다.
공식 테마 페이지 ↗ ko.wordpress.org/themes/astra
보유 라이선스

Avada

세계 1등 테마로 알려진 프리미엄 워드프레스 테마입니다. 지난번에 이미 구매하여 사용할 수 있는 라이선스가 있습니다.

  • 기존 구매 라이선스를 활용 가능
  • 완성도 높은 데모와 페이지 빌더 제공
  • 빠른 제작과 디자인 완성도 확보에 유리
이미 보유한 자산을 활용하므로 초기 선택 부담을 줄이고, 교회 홈페이지 개편을 빠르게 시작할 수 있습니다.

06 · Media Sharing

영상 및 사진 공유

영상은 자동 수집 후 담당자가 검수해 공개하고, 사진은 워드프레스에서 등록하되 파일은 외부 저장소와 NAS에 분리 보관합니다.

영상

  • 설교 사용자 정의 게시물 유형 생성
  • 예배별로 유튜브 재생목록 분리
  • YouTube Importer Pro로 새 영상 자동 등록
  • 처음 한두 달은 자동 공개하지 않고 초안 저장
  • 제목·설명·대표 이미지·영상 주소 자동 입력
  • 담당자가 성경 본문과 설교자만 확인한 뒤 공개

사진

  • 교회소식과 별도로 사진앨범 게시물 유형 생성
  • 관리자가 행사명과 날짜를 입력하고 사진을 한꺼번에 업로드
  • Offload Media 플러그인이 사진 파일을 R2에 자동 전송
  • 전송 확인 후 웹호스팅 서버의 원본 파일 삭제
  • 홈페이지에서는 일반 워드프레스 사진 게시판처럼 표시
  • 고해상도 원본은 현재 사용 중인 시놀로지 NAS에도 별도 백업
